Package com.devsoap.plugin.tasks

Package com.devsoap.plugin.tasks Description

Contains the tasks provided by the plugin.
  • Class Summary 
    Class Description
    BuildClassPathJar Builds a classpath jar used by external java processes.
    BuildJavadocJarTask Builds the Javadoc jar
    BuildSourcesJarTask Builds the sources jar
    CompileThemeTask Compiles the SASS theme into CSS
    CompileWidgetsetTask Compiles the Vaadin Widgetsets
    CompressCssTask Compresses the theme styles with GZip
    CreateAddonProjectTask Creates a new addon project with two modules, one addon module and one demo modules which utilizes the addon
    CreateAddonThemeTask Creates a theme for an addon
    CreateComponentTask Creates a new Vaadin Component
    CreateCompositeTask Creates a new Vaadin Composite
    CreateDesignTask Creates a new Vaadin Design
    CreateDirectoryZipTask Creates a Zip archive that is compatible with Vaadin Directory
    CreateProjectTask Creates a new Vaadin Project
    CreateTestbenchTestTask Creates a Vaadin Testbench test
    CreateThemeTask Creates a new theme
    CreateWidgetsetGeneratorTask Creates the widgetset generated class
    DevModeTask Runs GWT DevMode
    DirectorySearchTask Searches for an addon in the Vaadin Directory
    RunTask Runs the application on a application server
    SuperDevModeTask Runs the GWT codeserver as well as runs the application in SuperDevMode mode.
    UpdateAddonStylesTask Updates the addon.scss file listing with addons styles found in the classpath
    UpdateWidgetsetTask Updates the GWT module XML file with correct imports
    VersionCheckTask Checks the plugin version for a new version
